Qtip


Qtip è un plugin scritto con la libreria javascript jQuery e permette l'integrazione di "consigli" all' interno di testi, a differenza dell' aToolTip permette l'integrazione di veri e propri contenuti HTML (esempio).
Chiaramente trattandosi di contenuti HTML è possibile integrare preview video come ad esempio il trailer di un film oppure la preview di un tutorial relativo all'utilizzo di un prodotto o ancor meglio un video messaggio promozionale. Per concludere, è possibile mostrare l'anteprima di un sito all'interno di una piccola thumbnail, un metodo alternativo per pubblicizzare un partner o semplicemente link di utilità.
